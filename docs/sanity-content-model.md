# Sanity Content Model (for separate Studio app)

This frontend expects the following document types and fields. Field names should match exactly so the GROQ query in `src/lib/portfolio-content.ts` resolves without changes.

## `siteSettings` (singleton)
- `brandName` (string)
- `resumeUrl` (url)
- `navItems` (array of objects)
  - `label` (string)
  - `path` (string)
  - `external` (boolean, optional)
- `socials` (array of objects)
  - `platform` (string; recommended values: `github`, `linkedin`, `instagram`)
  - `label` (string)
  - `url` (url)

## `homePage` (singleton)
- `introTitle` (string)
- `introSubtitle` (string)
- `techSectionTitle` (string)
- `techStackRows` (array of objects)
  - `label` (string, optional)
  - `reverse` (boolean, optional)
  - `items` (array of objects)
    - `name` (string)
    - `iconKey` (string, optional)
    - `borderClass` (string, optional)
    - `textClass` (string, optional)
    - `iconClass` (string, optional)
- `experienceSectionTitle` (string)
- `experiences` (array of objects) OR use separate `experienceItem` docs (see below)
- `educationSectionTitle` (string)
- `educationItems` (array of objects) OR use separate `educationItem` docs (see below)
- `spotifySectionTitle` (string)
- `spotify` (object)
  - `enabled` (boolean)
  - `topTracksLimit` (number, max 5 recommended)
  - `topTracksRange` (string: `short_term|medium_term|long_term`)
  - `featuredPlaylists` (array of objects)
    - `spotifyPlaylistId` (string)
    - `customLabel` (string, optional)
    - `order` (number, optional)
- `githubSectionTitle` (string)
- `github` (object)
  - `enabled` (boolean)
  - `username` (string)
  - `pinnedReposLimit` (number, max 6 recommended)

## `experienceItem` (optional separate docs; auto-read by frontend)
- `company` (string)
- `role` (string)
- `image` (image, optional)
- `imageAlt` (string, optional)
- `employmentType` (string, optional)
- `location` (string, optional)
- `startDate` (string or date serialized to string)
- `endDate` (string, optional)
- `isCurrent` (boolean)
- `summary` (text)
- `highlights` (array of strings)
- `techTags` (array of strings)
- `order` (number)

## `educationItem` (optional separate docs; auto-read by frontend)
- `school` (string)
- `degree` (string)
- `image` (image, optional)
- `imageAlt` (string, optional)
- `program` (string, optional)
- `startDate` (string, optional)
- `endDate` (string, optional)
- `honors` (string, optional)
- `description` (text, optional)
- `tags` (array of strings)
- `order` (number)

## `aboutPage` (singleton)
- `title` (string)
- `name` (string)
- `profileImageUrl` (url)
- `bioTitle` (string)
- `bioParagraphs` (array of strings)

## `projectsPage` (singleton)
- `title` (string)
- `description` (text)
- `ctaLabel` (string, optional)
- `ctaUrl` (url, optional)
- `projects` (array of project-like objects, optional fallback)

## `project` (recommended collection docs; auto-read by frontend)
- `title` (string)
- `slug` (slug, optional)
- `summary` (text)
- `image` (image, optional)
- `imageAlt` (string, optional)
- `techTags` (array of strings, optional)
- `repoUrl` (url, optional)
- `liveUrl` (url, optional)
- `demoUrl` (url, optional; YouTube or other demo link)
- `demoLabel` (string, optional; defaults to `Demo`)
- `featured` (boolean, optional)
- `status` (string, optional)
- `order` (number)

## `contactPage` (singleton)
- `title` (string)
- `description` (text)
- `email` (string, optional)
- `ctaLabel` (string, optional)
- `ctaUrl` (url, optional)
